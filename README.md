## 使用说明

1. 安装扩展
   - 打开Chrome浏览器
   - 访问 `chrome://extensions/`
   - 开启右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择本扩展的文件夹

2. 使用步骤
   - 点击扩展图标打开操作界面
   - 点击"选择图片"按钮上传图片
   - 等待图片处理完成
   - 点击"下载分割图片"按钮下载所有分割后的图片
   - 选择保存位置后，四个部分会自动下载
